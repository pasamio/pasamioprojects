Installing Migrators

There are a few ways to install migrator extensions. If you have only one or two extensions, then you can use the form below to upload them manually. However if you have access to the server you can upload the plugin files into their respective directories (plugins and tables). The system uses only files that exist on the file system so registration with a database system isn't required for a plugin to apply.
